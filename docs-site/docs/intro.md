# Welcome to Setra

Setra is an open multi-agent CEO/specialist platform that runs your favorite
coding CLI (Claude, Codex, Gemini, OpenCode, Cursor) as a fleet of accountable
agents with persistent memory, governance, and analytics.

## Why Setra

- **No API keys.** Use whichever CLI you already authenticated.
- **Org hierarchy.** A CEO agent hires and oversees specialists.
- **Memory.** Per-agent and cross-agent recall via a queryable memory store.
- **Governance.** Hire-agent, spend, and tool-use are board-gated.
- **Observability.** SSE-driven board with KPIs, run timelines, and analytics.
